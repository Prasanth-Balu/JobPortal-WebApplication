# CareerHub - Job Portal Web Application

CareerHub is a responsive Job Portal web application built with React JS. 
It allows users to search for jobs, view detailed job information, and apply for jobs through a simple and user-friendly interface.


## Features

- Display job listings dynamically using JSON Server
- Search jobs by title, company, or location
- Browse featured jobs
- View complete job details
- Apply for jobs using an application form
- Store job application data in JSON Server
- Responsive design for desktop, tablet, and mobile devices


## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- React JS
- React Router DOM
- Fetch API
- JSON Server
